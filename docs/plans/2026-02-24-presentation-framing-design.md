# Presentation Framing Design: "5 Conclusions That Would Kill Your Policy"

## Central Thesis

*"In global health, a wrong conclusion costs lives. Here are 5 that this data tempts you to make — and the evidence they'd be premature."*

## Delivery Structure

### Act 1: The Split-Screen Hook (2 min)
Open with a dual-reality comparison. Show side-by-side:
- **Left:** What real-world health data looks like (literature benchmarks: access-mortality r ~ -0.45, 10-20pp country gaps, clear urban-rural disparities)
- **Right:** What THIS dataset shows (r ~ 0.0005, < 0.5pp gaps, identical country profiles)

Key visual: the "Expected vs Observed" paired bar chart from Q6. One slide, immediate impact.

Tagline: *"Before we draw conclusions, we need to check if our data can support them."*

### Act 2: The 5 Traps (8-10 min, ~2 min each)

Each trap follows the same 4-beat structure:

| Beat | Content |
|------|---------|
| **Common Claim** | The headline a naive analyst would write |
| **Our Evidence** | The statistical proof it's premature |
| **Why It's Premature** | Root cause (synthetic data / uniform distributions / no causal structure) |
| **Safer Decision Frame** | What a responsible analyst would say instead |

#### Trap 1: "Rural areas have worse health outcomes"
- Claim: Urbanization tier predicts mortality
- Evidence: Cohen's d < 0.01 for all tier pairs; eta-squared ~ 0
- Why premature: ANOVA p < 0.001 with N=1M, but effect is literally zero
- Safer frame: "We cannot distinguish outcomes across tiers with this data"
- **Key visual:** Effect size dot plot (Q1)

#### Trap 2: "Healthcare access improves outcomes"
- Claim: More access = lower mortality
- Evidence: Pearson r = 0.0005; R-squared < 0.001; flat decile chart
- Why premature: No dose-response relationship at any threshold
- Safer frame: "This data shows no access-outcome gradient — investigate data collection before reallocating resources"
- **Key visual:** Scatter + regression line showing the cloud of nothing (Q2)

#### Trap 3: "Country X is an outlier worth studying"
- Claim: Russia/Turkey defy expectations
- Evidence: Highest z-scores, but absolute differences < 0.5pp
- Why premature: Statistical significance ≠ practical significance at N=1M
- Safer frame: "No country deviates meaningfully — between-country variance is noise"
- **Key visual:** Quadrant scatter with labeled countries (Q3)

#### Trap 4: "Our tier definitions affect conclusions"
- Claim: Binary vs 3-tier vs 4-tier gives different answers
- Evidence: Max spread < 0.04pp across all 3 schemes; eta-squared ~ 0 for all
- Why premature: Conclusions identical regardless of classification
- Safer frame: "Robustness confirmed — but only because there's no signal to be sensitive TO"
- **Key visual:** 3-panel bar chart (Q4)

#### Trap 5: "The data supports resource reallocation"
- Claim: We can identify where to redirect health spending
- Evidence: All groups identical; no confounders; Shannon entropy near maximum
- Why premature: No signal for WHERE to reallocate — data is indistinguishable from random
- Safer frame: "Before any reallocation, validate data sources and collect targeted qualitative data"
- **Key visual:** Correlation matrix heatmap showing near-zero off-diagonal (Q6)

### Act 3: The Policy Guardrails (2-3 min)

Close with a "Minimum Data Requirements Before Policy Action" framework:

1. **Correlation sanity check:** Do expected variable relationships exist? (Income → Access, Access → Outcomes)
2. **Effect size thresholds:** Cohen's d > 0.2 before flagging disparities
3. **Confounder audit:** Are income/education confounding access → outcome?
4. **Power ≠ signal:** Large N doesn't mean meaningful differences exist
5. **Simulated uncertainty:** What happens to your CIs if you lose 95% of data?

Key visual: Evidence quality scorecard (Q6) — traffic-light heatmap.

Final slide: *"The most valuable analysis is the one that knows when NOT to make a recommendation."*
